# Blinky Command Guide

This guide outlines the standard behaviors and navigation paths for the Blinky command interface.

## ⚙️ Standard Behaviors

*   **Command Execution:** The primary function is executing commands.
*   **Text Manipulation:** Supports standard clipboard operations (Copy/Paste) within the command line interface.
*   **Platform Details:** This application runs on Windows/Linux environments.

## 🧭 Navigation & Shortcuts

*   **Settings:** `Ctrl + ,` (Opens Settings panel).
*   **Help:** `F1` (Opens the Help menu).

### ⌨️ Essential System Shortcuts

*   **Select All:** `Ctrl + A` (Selects all text on the current line/buffer).
*   **Copy:** `Ctrl + C` (Copies selected text to the clipboard).
*   **Paste:** `Ctrl + V` or `Shift + Insert` (Pastes copied text).
*   **Activate Menu Bar:** `F10` (Activates the menu bar in the active window).

### 💡 General Navigation

*   **Help/Support:** For general assistance, refer to the Support documentation or use the built-in search function.
*   **Taskbar/System:** Utilize system shortcuts to access the taskbar and system settings.

## 🚀 Productivity Tips

*   **Efficiency:** Relying on keyboard shortcuts (like `Ctrl + A` and `Ctrl + C`) is highly recommended for boosting productivity and speed within the command environment.

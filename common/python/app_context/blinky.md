# Blinky Command Guide

This guide outlines the standard behaviors and navigation paths for the Blinky command interface, focusing on efficient use of Windows system shortcuts.

## ⚙️ Standard Behaviors

*   **Command Execution:** The primary function is executing commands within the Windows environment. 
*   **Text Manipulation:** Supports standard clipboard operations (Copy/Paste) within the command line interface.

## 🧭 Navigation & Shortcuts

### ⌨️ Essential Windows Shortcuts

*   **Select All:** `Ctrl + A` (Selects all text on the current line/buffer).
*   **Copy:** `Ctrl + C` (Copies selected text to the clipboard).
*   **Paste:** `Ctrl + V` or `Shift + Insert` (Pastes copied text).
*   **Activate Menu Bar:** `F10` (Activates the menu bar in the active window).

### 💡 General Windows Navigation

*   **Help/Support:** For general Windows assistance, refer to the Microsoft Support website or use the built-in Windows search function.
*   **Taskbar/System:** Utilize standard Windows shortcuts (e.g., `Win` key) to access the taskbar and system settings.

## 🚀 Productivity Tips

*   **Efficiency:** Relying on keyboard shortcuts (like `Ctrl + A` and `Ctrl + C`) is highly recommended for boosting productivity and speed within the command environment.

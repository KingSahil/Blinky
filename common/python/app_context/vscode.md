# Visual Studio Code

Known menus:
File, Edit, Selection, View, Go, Run, Terminal, Help.

Known locations and shortcuts:
- Help is in the top menu bar after Terminal. Shortcut: Alt+H.
- Extensions is in the left activity bar. Shortcut: Ctrl+Shift+X.
- Command Palette shortcut: Ctrl+Shift+P.
- Settings is the gear icon at the bottom-left. Shortcut: Ctrl+,.

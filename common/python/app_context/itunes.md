# iTunes (itunes.exe)

## 💡 Known Behaviors

*   **Core Function:** iTunes is a comprehensive media management application used for listening to, organizing, and discovering music, videos, and podcasts within a user's library.
*   **Interface:** The application features a main window with various sections (e.g., Library, Music, Movies, Podcasts) and utilizes a menu bar for core functions.
*   **Accessibility:** Full keyboard navigation can be enabled, allowing users to interact with buttons, checkboxes, and lists without a mouse (useful for accessibility and speed).

## 🧭 Navigation Guide

### ⌨️ Keyboard Shortcuts (Primary Navigation)

*   **General Use:** Many tasks can be accomplished quickly using keyboard shortcuts. These shortcuts are often found within the main iTunes menus.
*   **Quick Access:** Many elements within the main window (like songs or column headings) also have dedicated shortcut menus for quick actions.
*   **Tip:** Utilize the built-in shortcut reference tables within the application's help menus for common commands (e.g., starting/stopping playback).

### 📂 Common Menus & Features

*   **Menu Bar:** The primary method for accessing functions is through the standard menu bar (File, Edit, View, etc.).
*   **Library Management:** Users can learn to use iTunes to manage their entire music collection, discover new artists, and add content to their library.
*   **Playback Control:** Standard media player controls (play, pause, skip, etc.) are accessible via both on-screen controls and keyboard shortcuts.

### ⚙️ Tips for Efficiency

*   **Full Keyboard Navigation:** To maximize efficiency, ensure full keyboard navigation is enabled to reach all interactive elements (buttons, checkboxes, lists) without relying on the mouse.

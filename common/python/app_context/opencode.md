# OpenCode Application Guide

## Standard Behaviors
- OpenCode uses a leader key for many keybinds to avoid conflicts, default is `ctrl+x`.
- Supports common Readline/Emacs-style text editing shortcuts in the prompt input box.

## Navigation Paths
### General Navigation
- Open OpenCode in a split terminal view or focus an existing session: `Cmd+Esc` (Mac) or `Ctrl+Esc` (Windows/Linux).
- Leader key + `n`: Start a new session.

### Menus and Shortcuts
- **Leader Key**: `ctrl+x` by default.
- **Common Actions**:
  - New session: Leader key + `n`.
- **Navigation Pane**: `Ctrl+O`.
- **Property Sheet**: `F11`.
- **Edit Mode**: Switch with `F4`.
- **Help**: `Ctrl+F` or use the Windows Menu (`Shift+F10`).

### Accessing Settings and Support
- Specific settings navigation not provided; assume standard Windows application settings access applies.
- For detailed help, refer to OpenCode's official documentation or support resources.

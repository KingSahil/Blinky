# wayland - linux desktop

**Standard Behaviors:**
- wayland is a display server protocol, not an application with a traditional GUI.
- It provides a Linux desktop environment, similar to GNOME or KDE.

**Navigation Paths:**
## Keyboard Shortcuts
- **Desktop Navigation:**
  + Alt + F1: Open Applications menu
  + Alt + F2: Open Run Application dialog
- **Window Management:**
  + Alt + Tab: Switch between open windows
  + Ctrl + Alt + Tab: Switch between open windows (alternative)
- **Settings:**
  + GNOME: Go to Settings via Applications menu or press `Super` key (Windows key equivalent)
## Accessing Menus
- **Applications Menu:**
  + Click Applications icon on the dock (if available)
  + Press Alt + F1
- **System Menu:**
  + Click System icon on the dock (if available)
  + Press `Super` key and navigate to System Settings
## Help and Support
- **GNOME Help:**
  + Press `Super` key and search for Help
  + Visit Official Ubuntu Documentation for more information

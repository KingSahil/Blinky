# Mozilla Firefox Navigation Guide

## Standard Behaviors
- Firefox uses standard Windows and browser conventions for navigation.
- Keyboard shortcuts are available for many tasks.

## Navigation Paths
### Menus and Shortcuts
- **File**: Access File menu by clicking the menu button (three horizontal lines) or pressing **Alt + F**.
- **Edit**: Access Edit menu by clicking the menu button or pressing **Alt + E**.
- **Help**: Open Help by pressing **F1** or clicking the menu button and selecting **Help**.

### Common Shortcuts
- **New Tab**: Ctrl + T
- **New Window**: Ctrl + N
- **Close Tab**: Ctrl + W
- **Undo Close Tab**: Ctrl + Shift + T
- **Refresh**: F5 or Ctrl + R

### Settings and Support
- **Settings**: Click the menu button, then select **Options** or press **Ctrl + Shift + A** and click **Options**.
- **Support**: Click the menu button, then select **Help** and choose from the available support options.

## Tips
- Use **Ctrl + Shift + T** to reopen closed tabs.
- Press **F5** to refresh the current page.
- Use **Ctrl + 1-8** to switch between tabs.

# Internal Console Management Window (openconsole.exe)

**Standard Behaviors:**
- The application window is titled 'internal console management window'
- Console output is displayed within the main window

**Navigation Paths:**
- **Settings:**
  * Click on 'File' > 'Settings' to access configuration options
- **Help/Support:**
  * Click on 'Help' > 'Documentation' for user guides
  * Click on 'Help' > 'About' for version information
- **Common Menus:**
  * 'File' menu: Exit, Settings
  * 'Edit' menu: Copy, Paste, Select All
- **Shortcuts:**
  * Ctrl+C to copy
  * Ctrl+V to paste
  * Ctrl+A to select all text

# Brave Browser

**Standard Behaviors:**
- Brave is a web browser based on Chromium.
- It focuses on privacy and security features.

**Navigation Paths:**
## Menus and Settings
- **Settings:** Navigate to `brave://settings/` in the address bar.
- **Bookmarks:** Accessible via the bookmarks bar or `brave://bookmarks/`.
- **Extensions:** Manage through `brave://extensions/`.

## Common Shortcuts
- **New Tab:** Ctrl + T
- **Close Tab:** Ctrl + W
- **Find:** Ctrl + F

## Help and Support
- **Help:** Use `brave://help/` for support and documentation.

## Additional Tips
- Use the address bar for quick searches or navigating to websites.
- The menu (three horizontal lines) provides access to bookmarks, downloads, and more.

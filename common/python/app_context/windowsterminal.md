# Windows Terminal (windowsterminal.exe)

**Standard Behaviors:**
- Opens to a command prompt interface.
- Title bar indicates 'administrator: command prompt'.

**Navigation Paths:**
## Menus and Settings
- **File Menu:**
  * Open new tabs/windows.
  * Access settings.
- **Settings:**
  * Can be accessed via `Settings` in the dropdown menu or by typing `settings` in the command line.
## Help and Support
- **Help:**
  * Type `help` in the command line for command-specific help.
  * Access documentation online via the dropdown menu or by typing `docs` in the command line.
## Common Shortcuts
- **New Tab:** Ctrl+Shift+`
- **Copy:** Ctrl+C
- **Paste:** Ctrl+V
- **Find:** Ctrl+F

**Additional Tips:**
- Use the dropdown menu to access settings, help, and other features.
